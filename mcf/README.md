## Возможности

– Подсветка ключевых слов: `if`, `else`, `while` и т. д.  
– Подсветка комментариев `# комментарий`.  
– Подсветка числовых литералов, строк.  
– Задача «Run MyLang Interpreter» для запуска `interpreter.py` на текущем файле.
